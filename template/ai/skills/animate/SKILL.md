# Animate

Read `../impeccable/reference/animate.md` completely and follow it.

This skill is a lightweight router. The canonical animation guidance stays in
Impeccable so it can be updated from its upstream source without maintaining a
second copy.

Do not use animation as decoration alone. Preserve reduced-motion behavior,
accessibility, responsiveness, and performance.
